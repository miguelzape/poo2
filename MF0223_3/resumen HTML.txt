Lista de los atributos más usados en HTML. Las etiquetas HTML pueden llevar estos atributos

    Id =" ": Para identificar un elemento único.
    class =" ": para Identificador de varios elementos.
    style =" ": Para darle un estilo al elemento.
	<p style="text-align: center;">Este es un párrafo alineado al centro.</p>
    alt=" ": Proporciona un texto alternativo para elementos multimedia, como imágenes.
	<img src="imagen.jpg" alt="Descripción de la imagen">
    href =" ": Para establecer URL de destino de un enlaces.
    height =" ": Para determinar altura.
    width =" ": Para determinar ancho.
    src=" ": Para especificar la fuente de imágenes.

Imagenes. Etiqueta <img/>

     Sus atributos

	src: Para indicar la ubicación o dirección de la imagen
		<img src="URL o nombre del archivo"/>
		<img src="koala.jpg"/>
		<img src="https://cdn.pixabay.com/photo/2015/02/09/20/03/koala-630117__340.jpg"/>
	alt: texto alternativo, este texto se mostrará si por alguna razón no carga la imagen.
		<img src="gatito.jpg" alt="foto de gatito"/>
	width: Ancho de imagen
	height: Alto de imagen
		<img src="koala.jpg" height="50px" width="150px"/>
		<img src="koala.jpg" height="50%" width="150%"/>
	vspce:	colocar espacio superio e inferior. se puede expresar en píxeles u otra unidad como em, %, pt, etc.
	border: para poner un marco en la imagen
		<img src="koala.jpg" border="10px" height="250px" width="400px"/>

Lista de terminologías o lista de definiciones 


    ETIQUETA <dl>.- Esta etiqueta se utiliza para crear una lista de definiciones si lo deseamos.
    ETIQUETA <dt>.- Esta etiqueta contiene el término que queremos definir.
    ETIQUETA <dd>.- Esta etiqueta contiene la definición del término que le precede.

		<dl><dt>Mamifero</dt><dd> Especie de animales que maman</dd></dl>
		<dl><dt>Vertebrado</dt><dd> Especie de animales que poseen huesos</dd></dl>
		<dl><dt>Reptil</dt><dd> Especie de animales que se arrastran por el suelo</dd></dl>

Tablas
	Etiqueta <table>: tabla
	atributo border: añade un borde a la tabla, que por defecto no tiene.
	<table border="1px"> 
		Etiqueta <tr>:fila (row)
			Etiqueta <td>: columna NORMAL dentro de la fila.
			Etiqueta <th>: columna IMPORTANTE dentro de la fila. se usa para encabezado (header)
			Atributo colspan y rowspan: extiende (o combina) la columna o fila el numero de casillas indicado
			<td rowspan="2"> ELEMENTO </td>
			<td colspan="2"> ELEMENTO </td>

	Etiquetas <thead>, <tbody> y <tfood> para tablas 
			<thead bgcolor="#88b1f7" height ="40px">
			<tr>
				<td colspan="6"> MI HORARIO DE CLASES </td>
			</tr>
  			</thead>

			<tfoot bgcolor="#88b1f7">
   			 <tr>
				<td colspan="6"> Horario verano del 5to B  </td>
			</tr>
 			</tfoot>

Formularios
	Etiqueta <form>: tabla
	Atributo action: será la URL de la página a la que se envia el formulario
	Atributo method: puede tomar como valores a GET o POST. y esto indica el método http que será utilizado.
	Atributo name: le pone nombre al formulario. Necesario cuando hay mas de un formulario en la misma pagina
	Atributo onsubmit: ejecuta funcion de validacion antes del summit. [onsummit="return validarForm()"]
			   con el return, solo envia formulario si validarForm devuelte true.
			   sin el return, envia el formulario despues del 'validarForm'. Sin inportar que devuelve
			
	Etiqueta <input>: campos del formulario
	ver:          https://developer.mozilla.org/es/docs/Web/HTML/Element/Input
	Atributo type: especifica el tipo de dato de entrada
   		Text: para campos de texto
			<input type="text" value="" placeholder="textom">: 
				placeholder muestra el 'textom' en la casilla para indicar su uso
				value pone un valor en la casilla. en este caso lo borra.
				tabindex="1" establece el orden de tabulacion de cada control
    		Password: para campos de contraseña
				<input type="password"/>	
    		Checkbox: para casillas de verificación
    		Radio: para casillas de selección
    		Submit: para botones de envío
    		Reset: para botones de resetear
    		File: para campos de selección de archivo
		tel: para telefono
	Etiqueta <select>: Crear una desplegable con opciones
	Atributo size.- Utiliza el atributo size para determinar la cantidad de opciones a mostrar y su valor en números.
	Atributo multiple.- el atributo multiple permite seleccion multiple de opciones 

		Etiqueta <option>: Opciones del desplegable

	    </form>
	Etiqueta <textarea>: Un campo de texto con varias lineas 
		<textarea name="mensaje"> </textarea>
